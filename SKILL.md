# storm-blindspot · 暴雨盲区

回答一个反直觉的问题：**我们用来研究洪水、规划排水、给房子定保费的那套"权威降雨数据"，到底把最凶的暴雨抹平了多少倍？**

再分析数据（ERA5 是全球气候研究和水文规划的通用底料）分辨率约 25km，它把一个城市格点里的降雨做了空间平均——于是把最致命的"局地对流暴雨"抹平了。这个工具用真实接口，把这个平时看不见的盲区**量化**出来。

## 何时使用

- 用户问"我家/这个小区这次暴雨会不会淹"
- 用户问某城市排水管按"几年一遇"设计、能扛多大雨
- 用户质疑或想验证 ERA5 / 再分析 / 网格气象数据靠不靠谱
- 用户想知道郑州7·20、深圳9·7、北京7·21那种极端雨"多少年一遇"
- 用户想把暴雨预警信号（蓝黄橙红）翻译成具体毫米数
- 用户想综合判断某地内涝风险（天上的雨 + 河里的水 + 脚下的洼地）

## 怎么用

```bash
cd <本 skill 目录>

python3 run.py calibrate                 # 核心：三城实测 vs ERA5，量出低估倍数 κ
python3 run.py blindspot zhengzhou       # 拉郑州40年ERA5，拟合"N年一遇"，对比实测看盲区
python3 run.py blindspot --lat 30.6 --lon 114.3 --years 44   # 任意坐标体检
python3 run.py warn shenzhen             # 用国标暴雨预警阈值翻译未来3天预报（绝对阈值，可信）
python3 run.py risk beijing              # 综合：降雨预警 + 上游河道流量 + 洼地指数
python3 run.py spec zhengzhou            # 该市坐标/分级/GB50014排水设计重现期/实测锚点
python3 run.py ls                        # 城市库（19座，3座带官方实测锚点）

# 任意子命令加 --json 输出结构化结果，喂给 agent
python3 run.py calibrate --json
```

纯标准库、零第三方依赖。唯一外部依赖是 open-meteo 的三个免费开放接口（archive/flood/elevation），无需 API Key。数学层（极值拟合、κ 标定）离线可测：`python3 tests/selftest.py`。

## 怎么读结果

- **低估倍数 κ = 官方实测 ÷ ERA5 峰值**：κ 越大，再分析数据把这场雨抹得越平。
  实测标定显示（同场同点）：郑州约 9.3×、北京约 4.2×、深圳约 3.4×。更狠的是，
  郑州实测 201.9mm/h 落在 ERA5 拟合的"千年一遇"（约54mm/h）之外，被当成"几乎不可能发生"。
- **关键结论**：κ 在不同城市差一个数量级，**不存在一个通用"订正系数"能把再分析数据修回真实**——
  盲区的宽度本身随地形和天气系统剧烈变化。这是本工具最想让你记住的一句话。
- **N 年一遇小时雨强**：注意这是"ERA5 口径"，被系统性压低。真实极端事件常常落在
  它拟合出的"千年一遇"之外——不是雨反常，是数据的分辨率盖不住它。
- **暴雨预警级别**：用的是绝对阈值（几小时下多少毫米），不受盲区影响，可直接采信。
- **内涝综合研判**：①降雨预警（可信）②河道流量趋势（参考）③洼地指数（地形参考）。

## 数据可信在哪

- ERA5 逐小时降雨来自 open-meteo archive-api（ECMWF ERA5/ERA5-Land 再分析）。
- 三个标定锚点均有官方出处，写在 `data/cities.json` 里，每条带来源：
  郑州 201.9mm/h（国务院灾害调查报告）、北京 100.3mm/h（中国气象局）、
  深圳 2h 195.8mm（深圳气象局/新华社）。
- 排水设计重现期来自 GB 50014-2021《室外排水设计标准》表4.1.3。
- 暴雨预警阈值来自中国气象局《气象灾害预警信号发布与传播办法》。

## 边界与免责

- 本工具**不是官方预警的替代品**。真正的内涝还取决于本地管网、泵站、实时调度，这些它看不见。
- κ 标定用的是"城市坐标所在 ERA5 格点值 vs 站点实测值"，两者空间代表性不同，
  倍数是**盲区量级的示意**，不是逐点精确的订正系数——恰恰相反，工具的结论就是"别信单一订正系数"。
- 极值拟合在样本年数少、或极端事件极为罕见时，尾部外推不确定性很大，仅供量级参考。
- 它替代不了官方预警——它替代的是"我以为这里不会淹"的那种错觉。
